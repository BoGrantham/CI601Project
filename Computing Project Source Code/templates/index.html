Welcome to my CI601 Computing Project Website!

This website was designed for the CI601 Computing Project module, the final project for all Computer Science degrees. My initial goal when creating this website was to foster an easily accessible space for others to connect, and I hope that the effort that I put into this project is shown.

On this website, you can use the discussion forum (after creating an account) to chat and make friends with others. You can also browse the advice section of the site.

Have fun exploring!
